To test built this project,

Go to your Processing Librarys directory and creat the sub directories .\pGui\library

you should now have a structure like this
C:\Users\user\Documents\Processing\libraries\pGui\library\
or
~/Processing/libraries/pGui/library/


Import the project into NetBeans

Edit the properties file nbproject/project.properties and change the dist.jar
property to your processing libraries path.

Like this
C:\Users\user\Documents\Processing\libraries\pGui\library\pGui.jar
or
~/Processing/libraries/pGui/library/pGui.jar
